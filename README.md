# FlowSuite

An integrated suite of lightweight web applications designed to streamline business operations, financial tracking, and team management.

## 🚀 Modules
- **FlowTrack**: Operations pipeline for logistics and project tracking.
- **FlowLedger**: Financial accounting and ledger reconciliation.
- **FlowPeople**: Human resources, payroll, and performance reviews.

## 🛠️ Tech Stack
- **Frontend**: Vanilla JavaScript, HTML5, CSS3 (Tailwind-style variables).
- **Backend**: Supabase (PostgreSQL, Auth, Real-time).
- **Automation**: EmailJS for status notifications.

## 📦 Deployment
This suite is designed to be hosted via GitHub Pages or any static web server.
